# Handoffs

Ephemeral "you are here" notes between work sessions live here as `HANDOFF.md`
(written/resumed by the `creating-handoffs` skill). Each holds the current
session's state: what's in flight, what's half-done, and the commands to verify
it — not durable docs.

There is no active handoff right now. When one exists it will be `HANDOFF.md` in
this directory.

Durable knowledge does **not** belong here — route it by lifespan:

- Architecture / data flows → [`../../ARCHITECTURE.md`](../../ARCHITECTURE.md)
- Topic references → [`../index.md`](../index.md)
- New features / larger efforts / backlog → [`../../ROADMAP.md`](../../ROADMAP.md)
- In-scope tangents to clear before the next feature → [`../../FOLLOWUPS.md`](../../FOLLOWUPS.md)
- Shipped changes → [`../../CHANGELOG.md`](../../CHANGELOG.md)
